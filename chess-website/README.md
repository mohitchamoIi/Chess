# Chess Website
A interactive web-based chess game built with vanilla JavaScript, HTML, and CSS.

## Features

- Play chess against an opponent
- Interactive board with drag-and-drop piece movement
- Real-time move validation
- Move history tracking
- Responsive design for desktop and mobile devices

## Project Structure

```
chess-website/
├── index.html      # Main HTML file with game board
├── style.css       # Styling for the chess board and pieces
├── script.js       # Game logic and piece movement
└── README.md       # Project documentation
```

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, or Edge)

### Installation

1. Clone or download the project files
2. Open `index.html` in your web browser
3. Start playing!

## How to Play

1. Click on a piece to select it
2. Available moves will be highlighted
3. Click on a highlighted square to move the piece
4. The game enforces all standard chess rules
5. Players alternate turns (White moves first)

## Game Rules

- Standard chess rules apply
- Pieces move according to their type
- Check, checkmate, and stalemate detection
- En passant, castling, and pawn promotion support

## Technologies Used

- **HTML5** - Structure and board layout
- **CSS3** - Styling and animations
- **JavaScript (ES6+)** - Game logic and interactivity
